## Getting Help

### 1. Search Existing Issues

Someone may have already asked your question:

- Search [GitHub Issues](https://github.com/neurova/neurova/issues)
- Check [GitHub Discussions](https://github.com/neurova/neurova/discussions)

### 2. Ask a Question

If you can't find an answer:

**For usage questions**:

- Open a [GitHub Discussion](https://github.com/neurova/neurova/discussions)
- Use the Q&A category

**For bugs**:

- Open a [GitHub Issue](https://github.com/neurova/neurova/issues/new)
- Use the bug report template

**For feature requests**:

- Open a [GitHub Issue](https://github.com/neurova/neurova/issues/new)
- Use the feature request template

### 3. Provide Context

When asking for help, include:

- **What you're trying to do**
- **What you've already tried**
- **Error messages** (full traceback)
- **Environment details** (Python version, OS, Neurova version)
- **Minimal code** to reproduce the issue

Example:

```python
import neurova as nv

# This doesn't work as expected:
result = nv.some_function(input_data)
# Error: ...
```

- **Questions**: Usually answered within a few days
- **Feature requests**: Discussed as time allows
