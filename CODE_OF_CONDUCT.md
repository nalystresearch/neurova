# Code of Conduct

## Our Commitment

We are committed to providing a welcoming and respectful environment for everyone who contributes to or uses Neurova, regardless of background, experience level, identity, or perspective.

## Expected Behavior

All community members are expected to:

- **Be respectful**: Treat others with courtesy and respect
- **Be collaborative**: Work together constructively and help each other
- **Be professional**: Keep discussions focused and productive
- **Be inclusive**: Welcome newcomers and help them get started
- **Be open-minded**: Accept feedback and different viewpoints gracefully
- **Be patient**: Remember that everyone is learning and growing

## Unacceptable Behavior

The following behaviors are not tolerated:

- **Harassment**: Any form of harassment, intimidation, or unwelcome behavior
- **Discrimination**: Discrimination based on personal characteristics
- **Disrespect**: Insulting, demeaning, or attacking others
- **Trolling**: Deliberate disruption of discussions or project work
- **Privacy violations**: Sharing others' private information without consent
- **Spam**: Irrelevant or commercial messaging

## Scope

This Code of Conduct applies to:

- Project repositories and pull requests
- Issue trackers and discussions
- Project communication channels
- Project events and meetups
- Any space representing the project

## Reporting

If you experience or witness behavior that violates this Code of Conduct:

1. **Document the incident**: Save relevant messages or screenshots
2. **Report to maintainers**: Contact project maintainers privately
3. **Provide details**: Include what happened, when, and who was involved

Contact: For serious violations, email the project maintainers

## Enforcement

Project maintainers are responsible for enforcement and may:

1. **Issue a warning**: First-time or minor violations
2. **Request changes**: Ask for edits to comments or contributions
3. **Temporary suspension**: Restrict access for repeat violations
4. **Permanent ban**: Remove from the community for severe or repeated violations

All reports will be reviewed fairly and confidentially.

## Attribution

This Code of Conduct is based on best practices from open source communities and reflects our commitment to maintaining a positive environment for all contributors.

## Questions

If you have questions about this Code of Conduct or need clarification, please open an issue or contact the maintainers.

---

**Remember**: We're all here because we care about building great software. Let's treat each other well and focus on making Neurova better together.
